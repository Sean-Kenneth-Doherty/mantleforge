## Problem & Solution

### Problem
Creating smart contracts requires specialized Solidity knowledge, creating barriers for:
- Non-technical founders launching tokens
- Developers new to Web3
- Teams needing rapid prototyping

### Solution
MantleForge uses AI to translate natural language into production-ready Solidity contracts deployable to Mantle Network.

**Example:**
Input: "Create an ERC-20 token called MantleGold with 1 million supply"
Output: Complete, deployable ERC-20 contract

3. **Fast finality** - Quick deployments

## Demo Instructions

1. Visit https://sean-kenneth-doherty.github.io/mantleforge/
2. Click "Connect Wallet" (adds Mantle Sepolia automatically)
